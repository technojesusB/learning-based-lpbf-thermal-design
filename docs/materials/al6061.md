# Aluminum Alloy 6061

High-strength aluminum alloy widely used in structural applications.

## Key Physical Constants
| Property | Value | Source |
| :--- | :--- | :--- |
| **Solidus Temperature ($T_{sol}$)** | 582 °C (855 K) | [1] |
| **Liquidus Temperature ($T_{liq}$)** | 652 °C (925 K) | [1] |
| **Latent Heat of Fusion ($L_f$)** | 400 J/g | [1] |
| **Reference Density ($\rho_{RT}$)**| 2700 kg/m³ | [1] |

## Simulation Assumptions & Extrapolation
*   **Solid Behavior**: Strong temperature dependence in $C_p$ above 400 °C.
*   **Melt Stability**: Liquid properties capped at the liquidus value to prevent solver divergence.
