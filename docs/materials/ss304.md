# Stainless Steel 304 (SS304)

Austenitic SS304 is the baseline stainless steel for many industrial applications.

## Key Physical Constants
| Property | Value | Source |
| :--- | :--- | :--- |
| **Solidus Temperature ($T_{sol}$)** | 1400 °C (1673 K) | [1] |
| **Liquidus Temperature ($T_{liq}$)** | 1450 °C (1723 K) | [1] |
| **Latent Heat of Fusion ($L_f$)** | $\approx 268$ J/g | [1] |
| **Reference Density ($\rho_{RT}$)**| 8000 kg/m³ | [1] |

## Simulation Assumptions & Extrapolation
*   **Solid Range**: Linear extrapolation of experimental point data from room temperature up to $T_{sol}$.
*   **Liquid Range**: $k$ and $C_p$ assumed constant at the liquidus value for stability in the melt pool.
