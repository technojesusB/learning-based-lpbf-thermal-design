# Nickel Alloy 718 (Inconel 718)

High-strength, corrosion-resistant nickel-chromium superalloy.

## Key Physical Constants
| Property | Value | Source |
| :--- | :--- | :--- |
| **Solidus Temperature ($T_{sol}$)** | 1260 °C (1533 K) | [1] |
| **Liquidus Temperature ($T_{liq}$)** | 1336 °C (1609 K) | [1] |
| **Latent Heat of Fusion ($L_f$)** | 227 J/g | [1] |
| **Reference Density ($\rho_{RT}$)**| 8190 kg/m³ | [1] |

## Simulation Assumptions & Extrapolation
*   **Solid Extension**: Linear extrapolation of conductivity and specific heat from 1100K up to the solidus.
*   **Liquid Transition**: Heat capacity remains constant at 635 J/kgK above 1100K (simulated plateau).
