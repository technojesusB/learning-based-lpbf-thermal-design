# Nickel Alloy 625 (Inconel 625)

$\gamma$-strengthened nickel superalloy with excellent weldability and high-temperature strength.

## Key Physical Constants
| Property | Value | Source |
| :--- | :--- | :--- |
| **Solidus Temperature ($T_{sol}$)** | 1290 °C (1563 K) | [1] |
| **Liquidus Temperature ($T_{liq}$)** | 1350 °C (1623 K) | [1] |
| **Latent Heat of Fusion ($L_f$)** | $\approx 230$ J/g | [Est.] |
| **Reference Density ($\rho_{RT}$)**| 8440 kg/m³ | [1] |

## Simulation Assumptions & Extrapolation
*   **Data Limit**: Experimental data utilized up to 1255K (982 °C).
*   **Extrapolation Logic**: Constant $k, C_p$ assumed above 1255K through the melting range for computational robustness.
