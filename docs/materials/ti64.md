# Titanium Alloy Ti-6Al-4V (Grade 5)

High-performance $\alpha$-$\beta$ titanium alloy. Essential for aerospace and medical LPBF applications.

## Key Physical Constants
| Property | Value | Source |
| :--- | :--- | :--- |
| **Solidus Temperature ($T_{sol}$)** | 1604 °C (1877 K) | [2] |
| **Liquidus Temperature ($T_{liq}$)** | 1660 °C (1923 K) | [2] |
| **Latent Heat of Fusion ($L_f$)** | 286 J/g | [2] |
| **Reference Density ($\rho_{RT}$)**| 4430 kg/m³ | [1] |

## Simulation Assumptions & Extrapolation
*   **As-Built Solid ($T < 873$K)**: Uses LPBF-specific data from Akwaboa (2023) reflecting high cooling rates.
*   **Bulk Transition**: Transitions to Boivineau (2006) bulk property data above 1000K.
*   **Liquid Phase ($T > T_{liq}$)**: Linear extrapolation of conductivity; constant specific heat.
