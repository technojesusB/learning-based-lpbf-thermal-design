# 17-4PH Stainless Steel

Martensitic precipitation-hardening steel used for high strength and hardness.

## Key Physical Constants
| Property | Value | Source |
| :--- | :--- | :--- |
| **Solidus Temperature ($T_{sol}$)** | 1404 °C (1677 K) | [1] |
| **Liquidus Temperature ($T_{liq}$)** | 1440 °C (1713 K) | [1] |
| **Latent Heat of Fusion ($L_f$)** | $\approx 270$ J/g | [Est.] |
| **Reference Density ($\rho_{RT}$)**| 7800 kg/m³ | [1] |

## Simulation Assumptions & Extrapolation
*   **Transition Phase**: Solidification behavior is complex (delta-ferrite nucleation); handled as a smoothed latent heat peak between 1677K and 1713K.
*   **Liquid Phase**: Properties assumed stationary above liquidus.
