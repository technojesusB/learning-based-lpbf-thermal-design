# H13 Tool Steel

Hot-work tool steel with excellent thermal shock resistance. Commonly used for die-casting inserts.

## Key Physical Constants
| Property | Value | Source |
| :--- | :--- | :--- |
| **Solidus Temperature ($T_{sol}$)** | 1335 °C (1608 K) | [1] |
| **Liquidus Temperature ($T_{liq}$)** | 1475 °C (1748 K) | [1] |
| **Latent Heat of Fusion ($L_f$)** | 280 J/g | [1] |
| **Reference Density ($\rho_{RT}$)**| 7750 kg/m³ | [1] |

## Simulation Assumptions & Extrapolation
*   **Extension to Melt**: Conductivity is extended linearly from 1000K to the liquidus.
*   **Liquid State**: $k$ and $C_p$ remain constant above $T_{liq}$.
